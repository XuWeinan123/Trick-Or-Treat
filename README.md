# 🎃 Trick or Treat

这是一个充满趣味性的 Figma 恶作剧（Prank）插件！它会在你不注意的时候，给你（或你的同事）送上一份万圣节“惊喜”。

## 功能特点

- **设计转图**：自动将选中的 Frame 或 Component 转换为 PNG 图片。
- **保留容器**：保留原始的容器结构，但里面的内容已经变成了静态图片。
- **位置打乱**：随机调换这些被转成图片的节点位置，完成一次完美的“恶作剧”。

## 快速上手

1. 克隆本仓库到本地。
2. 在 Figma 中选择 `Plugins` -> `Development` -> `Import plugin from manifest...` 并选中 `manifest.json`。
3. 选中您要作弄的画布节点，运行插件，见证“奇迹”发生！
